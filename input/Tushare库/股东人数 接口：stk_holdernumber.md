
描述：获取上市公司股东户数数据，数据不定期公布
限量：单次最大3000,总量不限制
积分：600积分可调取，基础积分每分钟调取100次





**输入参数**

| 名称       | 类型 | 必选 | 描述         |
| :--------- | :--- | :--- | :----------- |
| ts_code    | str  | N    | TS股票代码   |
| ann_date   | str  | N    | 公告日期     |
| enddate    | str  | N    | 截止日期     |
| start_date | str  | N    | 公告开始日期 |
| end_date   | str  | N    | 公告结束日期 |





**输出参数**

| 名称       | 类型 | 默认显示 | 描述       |
| :--------- | :--- | :------- | :--------- |
| ts_code    | str  | Y        | TS股票代码 |
| ann_date   | str  | Y        | 公告日期   |
| end_date   | str  | Y        | 截止日期   |
| holder_num | int  | Y        | 股东户数   |
