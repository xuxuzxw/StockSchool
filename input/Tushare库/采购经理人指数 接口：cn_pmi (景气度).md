限量：单次最大2000，一次可以提取全部数据
权限：用户积累2000积分可以使用

**输入参数**

| 名称    | 类型 | 必选 | 描述                                                |
| :------ | :--- | :--- | :-------------------------------------------------- |
| m       | str  | N    | 月度（202401表示，2024年1月）                       |
| start_m | str  | N    | 开始月度                                            |
| end_m   | str  | N    | 结束月度（e.g. fields='month,pmi010000,pmi010400'） |
